projection.get_data(relays, guards=False)¶

input a list of relay objects and analyze the bandwith of coverage, invalid reason etc for all relays and record the percentage of each spec. The graphing functions visualizes specs calculated here.

Parameters

  • relays – (list) list of relay objects

  • guards – (bool) boolean specify whether to only analyze guard relay

Returns

(dict) spec -> value of spec

projection.get_guards(relays)¶

filter out guard relay from a list of relays

Parameters

relays – (list) list of TOR relay object

Returns

(list) list of TOR relay that has the guard flag

projection.main()¶

call the above graphing function at the end of main function to graph the projections of ROA coverage under different scenarios.
