util module¶

class util.Relay(fp, ip)¶

Bases: object

Relay Object to store information on each relay parsed from consensus file stores the relay’s ip, fingerprint etc.

util.datespan(start_date, end_date, delta=datetime.timedelta(seconds=3600))¶

Function to iterate through each hour in a given timespan and out put each iteraton

Parameters

  • start_date – (datetime object) start date of the duration

  • end_date – (datetime object) end date of the duration

  • delta – (timedelta) the interval of each iteration, default to 1 hour

util.get_all_ips(start_date, end_date)¶

get all ip addresses of relays in a duration of consensus files, draw data from the archive_pickle folder

Parameters

  • start_date – (datetime object) start date of the duration

  • end_date – (datetime object) end date of the duration

Returns

(set) return 2 sets containing ip address, for IPv4 and IPv6 addresses

util.get_prefix_and_asn(ip)¶

old method of getting prefix and asn given ip, currently uses routeview data and searches offline see get_prefix_and_asn_local(ip) for new method in preprocess_consensus.py

Parameters

ip – (str) ip address in string format

Returns

(list) returns a list containing prefix and ASN of the ip address

util.get_roas(filename)¶

parse roa file in .csv format, put each entry in a list, then append to a list containing all entries. each line in the file converts into [ipv4, maxlen, prefixlen, asn]

Parameters

filename – (string) name of the csv file to parse

Returns

(list) 2 lists of list for ipv4 and ipv6 addresses, entry in each list is [ipv4, maxlen, prefixlen, asn]

util.load_consensus(p, year, month, date, hour)¶

(same as the function in process_consensus.py) pulls list of relay object, weight info (wgd, wgg) from pickled file

Parameters

  • p – (str) path of pickle directory

  • year – (str) year of the consensus

  • month – (str) month of the consensus

  • date – (str) date of the consensus

  • hour – (str) hour of the consensus

Returns

list of relay objects, wgd, wgg
