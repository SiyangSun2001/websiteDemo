getASNipFreqOrigin module¶

getASNipFreqOrigin.getAllAS3(routeviewFile)¶

using routeview data, create dictionary ASn -> ASN object. ASN object has the AS’ prefixes and total number of IP addresses.

Parameters

routeviewFile – (string) file path to the routeview data

getASNipFreqOrigin.get_max_min(ipPrefix)¶

input an IP prefix and return the highest and lowest IP address

Parameters

ipPrefix – (string) IPv4 prefix

Returns

(IPv4Address object) return the lowerst address and highest address

getASNipFreqOrigin.get_origin(asn_orgID, orgID_origin, ASNnumIP)¶

assign origin to all ASes after tallying up the number of IP addresses announced

Parameters

  • asn_orgID – (dictionary) helper dict obtained from preprocess_asn_origin(caidaFile)

  • orgID_origin – (dictionary) helper dict obtained from preprocess_asn_origin(caidaFile)

  • ASNnumIP – (dictionary) result dict of all AS with number of Ip addresses tallied up using getAllAS3()

Returns

(dictionary) return ASNnumIP dict with the origin of each AS filled in

getASNipFreqOrigin.get_prefix_addresses_map()¶

return a map of IP prefix to number of possible users. used to quickly sum up the number of possible IPv4 announced within an AS

/32 -> 0 client /31 -> 2 clients

getASNipFreqOrigin.main()¶

main function that uses the above function to save a pickled dictionary containing ASes filled with origin and number of IP addresses it announces download the prefix to ASN file (routeviews-rv2-20210722-0200.pfx2as) here https://publicdata.caida.org/datasets/routing/routeviews-prefix2as/

getASNipFreqOrigin.preprocess_asn_origin(caidaFile)¶

preprocess the caida data to assign origin to ASN. map orginzation id to origin country https://www.caida.org/catalog/datasets/as-organizations/

Parameters

caidaFile – (string) path to the Caida file

Returns

(dictionary) 2 dictionary that when used together could find the origin of an AS using ASN
